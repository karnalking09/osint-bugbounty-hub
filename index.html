Email OSINT & Reputation

  • HaveIBeenPwned
    Breach lookup for email addresses
    Visit: https://haveibeenpwned.com
  • Emailrep.io
    Email reputation API (threat indicators)
    curl "https://emailrep.io/" -H "Key: YOUR_KEY"
  • h8mail
    Deep breach recon & password harvesting
    git clone https://github.com/khast3x/h8mail.git
cd h8mail && pip install -r requirements.txt
  • GHunt
    Gmail-focused OSINT (Drive, YouTube, photos)
    git clone https://github.com/mxrch/GHunt.git
# follow README for installation
  • Epieos
    Email/phone footprinting (web)
    Visit: https://epieos.com

Phone Number OSINT

  • PhoneInfoga
    Phone number recon (carrier, location)
    git clone https://github.com/sundowndev/PhoneInfoga.git
cd PhoneInfoga && pip install -r requirements.txt
  • Truecaller
    Caller ID & spam detection (app/web)
    Install app from Play Store / visit: https://www.truecaller.com

Username & Social Media Discovery

  • WhatsMyName
    Search username across many platforms
  • Sherlock
    CLI username search (~300 sites)
    git clone https://github.com/sherlock-project/sherlock.git
python3 -m pip install -r requirements.txt
  • Maigret
    Sherlock alternative with profile enrichment
    pipx install maigret
maigret username

Website, Domain & DNS Recon

  • OWASP Amass
    In-depth attack surface mapping & subdomain discovery
    go install -v github.com/OWASP/Amass/v3/...@latest
amass enum -d example.com
  • subfinder
    Fast passive subdomain discovery
    go install -v github.com/projectdiscovery/subfinder/v2/cmd/subfinder@latest
subfinder -d example.com -o subdomains.txt
  • dnsx
    High-performance DNS probing & resolution
    go install -v github.com/projectdiscovery/dnsx/cmd/dnsx@latest
cat subs.txt | dnsx -resp
  • massdns
    Bulk DNS resolver for fast lookups
  • SecurityTrails / ViewDNS
    WHOIS, reverse IP and DNS tools (web)

Quick install examples

Use depending on your OS. Many tools need Go, Python or Docker. 
# Go (recommended for many ProjectDiscovery tools)
# install Go 1.20+, then:
go install github.com/projectdiscovery/subfinder/v2/cmd/subfinder@latest

# Git clone + pip
git clone https://github.com/sherlock-project/sherlock.git
python3 -m pip install -r sherlock/requirements.txt

# apt (Debian/Ubuntu)
sudo apt update && sudo apt install nmap git python3-pip -y
